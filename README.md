Features 
--------
 * no Tango support 
 * cmake version >=2.8
 * Linux Platform first (DMD, GDC and LDC). 
 * Windows will come later
  
## Installation

* cd cmaked/cmaked
* mkdir build
* cd build
* cmake ..
* make install (with sudo to install in the system path)
